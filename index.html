The console object gives you access to the browser's console. It lets you output strings, arrays, and objects that help debug your code. The console is part of the window object, and is supplied by the Browser Object Model (BOM) . To open it first we have to right click on the live server then we will get to see a pop-up screen in which there is a option called "Inspect Element" we have to click that option then the console screen will appear .
